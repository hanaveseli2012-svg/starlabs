# StarLabs: Mission Innovation

An educational game where you progress from **Junior Developer** to **Lead Innovator** at StarLabs, a modern software company.

## Play

Open `index.html` in any browser — no install, no server needed.

## Levels

| Level | Name | Gameplay |
|-------|------|----------|
| 1 | Idea Hunt | Explore the office collecting floating lightbulbs (WASD/arrows) |
| 2 | Coding Challenge | 8 coding & logic questions — shuffled each play |
| 3 | Bug Hunter | Click to squash spreading bugs before time runs out |
| 4 | Client Rush | Pick the best tech solution for 6 client scenarios |
| 5 | Product Launch | Review your stats, then solve the **STARLABS** hangman puzzle! |

## Features

- XP system with 5 promotion levels
- 8 achievement badges
- Leaderboard (localStorage)
- Sound effects (Web Audio API)
- Particle effects & animations
- Computer avatar character
- Questions shuffle every playthrough

## Tech

Single-file HTML5 Canvas game — vanilla JavaScript, zero dependencies.
